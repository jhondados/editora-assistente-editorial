# 📚 Assistente Editorial com IA

> Assistente de IA generativa para editoras: avalia originais, sugere revisões, gera sinopses, palavras-chave e metadados, e estima potencial de mercado de novos manuscritos.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/editora-assistente-editorial/

## 📊 Resultados-chave (KPIs)
**10x** Triagem · **escala** Originais · **auto** Metadados · **78%** Acerto mercado

## 🧠 Tecnologias de IA
GenAI · LLM · RAG

## ✨ O que faz
- Triagem de originais
- Geração de sinopse e metadados
- Sugestão de revisão (GenAI)
- Estimativa de potencial
- Palavras-chave SEO de livro

## 🗄️ Fonte de dados real
[Manuscritos + catálogo + mercado](https://www.cbl.org.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Manuscritos + catálogo + mercado
2. **Processamento com IA** — GenAI · LLM · RAG
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
